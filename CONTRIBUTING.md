## System requirements

* [nodejs >= 8](https://nodejs.org/en/)
* [yarn](https://yarnpkg.org) (used for managing dependencies)
* [lerna](https://github.com/lerna/lerna/) (used for managing subprojects)

## Setup

After cloning the repository, run `lerna bootstrap` to install packages. After
that you can `cd` into the directory of the subproject you want to work on, and
follow the instructions in the subproject's **CONTRIBUTING.md**.
